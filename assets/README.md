# Assets Directory

This directory contains images and other assets for the GFN for Deck plugin.

## Required Assets

### Plugin Icon

- **File**: `logo.png`
- **Size**: 256x256px recommended
- **Format**: PNG with transparency
- **Usage**: Shown in Decky Loader plugin list and store

### Screenshots (for plugin store)

- **Files**: `screenshot1.png`, `screenshot2.png`, etc.
- **Size**: 1280x800px (Steam Deck native resolution)
- **Format**: PNG or JPEG
- **Usage**: Display in plugin store listing

## Adding Assets

1. Place image files in this directory
2. Update references in `plugin.json` if needed
3. For the plugin icon, use a high-quality GFN-inspired logo

## Design Guidelines

- Keep the icon simple and recognizable
- Use NVIDIA GeForce NOW brand colors:
  - Primary: #76b900 (green)
  - Secondary: #000000 (black)
- Ensure the icon is visible on both light and dark backgrounds
- Follow Steam Deck's UI design language
